Developing Machine Learning Model for Union College Admissions

Continuing my summer research project from the summer of 2020, I am also working with Union College's Admissions department to develop a machine learning model based on their application data. The goal of the model is to predict the probability of enrollment for a given applicant given the information they included when applying to the College, how many times they visited campus, etc. These predictions are, in turn, used to create a prospective class profile to forecast the enrollment and finances of an incoming class.

This project will be used as a reference point in the College's 2021 Admissions cycle, and my work this fall is focused on refining the model for use.
