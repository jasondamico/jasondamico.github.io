2020 Summer Research Project: Predicting Incoming Union College Class Profiles using Machine Learning

June-August 2020

In the summer of 2020, I conducted a research project through Union College to study the effectiveness of using various machine learning models to predict the probability of enrollment of Union applicants based on eight years of application data. Those probabilities are then used in aggregate to generate an expected class profile for the incoming class. The project was written in Python and primarily used the scikit-learn and pandas libraries to conduct data analysis.

To prepare for the project, I used Tableau for exploratory data analysis (EDA), which helped me understand which trends in the data might be helpful to the predictive model. I also read The Hundred-Page Machine Learning Book by Andriy Burkov to obtain exposure to the theory surrounding machine learning, then implemented what I read into the project.

I programmed the whole project myself, while receiving and implementing feedback from a Union Economics professor and a member of the Admissions staff as I developed the model. The work I did over the summer was successful: Admissions plans on using my model as a reference point for the 2021 applicant class. I am presently working to refine the model and prepare it for deployment. For more information on this project and an analysis of my findings from the summer, please visit the online publication of my summer research poster.
